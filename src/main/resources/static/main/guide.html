繁星沙岸

有「威尼斯海灘」之稱的蛤板灣，是堆積地形的貝殼沙岸，在這也有許多潮間帶地形，風平浪靜時，是著名玩水活動之處，傍晚駐足在沙灘，可在此欣賞海面上的日落美景。

島上觀賞日出及落日的絕佳之處，緊臨壯麗的海，隨著太陽緩緩、靜靜的上昇落下。

海成壺穴，再經過溶蝕作沉澱作用形成的特殊珊瑚礁岩洞，每當海浪進入溝中，潮聲如吼，浪濤千堆雪。來到綿延不絕的珊瑚裙礁，特別的海蝕平台，嶙峋的岩石遍佈其中，是海岸公路上，值得駐足的自然風光。

行程安排

報到出發(旅館) -> 海岸公路(天然洞穴) -> 午餐時間 -> 繁星沙岸 -> 快樂賦歸(旅館)

如何報名?

此套裝行程為旅行社與旅館合作推出之限定行程，請於訂房流程中選擇欲參加之行程，並確認開團日期與資訊


1.當天正確報到時間，以行程資料為主。
2.報名時請注意個人身體狀況是否適合參與戲水活動
3.沙灘環境請大家共同維護，勿帶過多包裝用品。
4.午餐自理(建議自備：巧克力、飯糰、麵包、乾糧)。

林間巡禮

位於尖石鄉，是台灣著名的原住民泰雅族部落，擁有《黑色部落》之稱，是全台最後通電的一個部落，因長年交通不便，造就出一個遺世獨立的部落和巨木生態的山川景緻和森林風貌。

這裡沒有水泥叢林，只有翠綠山林。 探訪台灣排名第三大的神木，其樹種為紅檜，
樹高約35公尺、 樹圍約20.5公尺，形狀類似燭台且須20人左右才可將神木環抱，
樹齡約2500年，為了保護神木樹根已在樹根部分建起木造平台； 在司馬庫斯的登山路線中，
除了可看到時代久遠的巨木群以外，楓樹、櫻花、竹林、 扁柏和紅檜林立，溪澗、
碎石落盤地質相鄰可見，是台灣珍貴的山林資源。



行程安排

報到出發(旅館) -> 司馬庫斯(部落) -> 午餐時間 -> 神木群探索 -> 快樂賦歸(旅館)

此套裝行程為旅行社與旅館合作推出之限定行程，請於訂房流程中選擇欲參加之行程，並確認開團日期與資訊


1.當天正確報到時間，以行程資料為主。
2.報名時請注意個人身體狀況是否適合參與登山運動
3.森林環境請大家共同維護，勿帶過多包裝用品。
4.午餐自理(建議自備：巧克力、飯糰、麵包、乾糧)。
